# Loan Repayment Prediction – ML Project

### Overview
A machine learning model to predict whether customers will repay a micro-loan within 5 days.  
This repository includes the complete code, final results, and a detailed PDF report.

### Models Used
- Logistic Regression  
- Decision Tree  
- Random Forest  
- Gradient Boosting  
- XGBoost (Best Performer)

### Best Model
**XGBoost**
- Accuracy: 94.47%
- Precision: 91.84%
- Recall: 87.82%
- F1 Score: 89.78%

### Files Included
- **Code#P1.ipynb** – Full machine learning code  
- **Project1.pdf** – Report  
- **Results.xlsx** – Final output
